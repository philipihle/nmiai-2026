# NM i AI 2026

Our solutions for [NM i AI 2026](https://ainm.no) — Norway's National AI Championship.

**Repository:** https://github.com/philipihle/nmiai-2026

## Tasks

### Task 1 — Tripletex (AI Accounting Agent)
`/tripletex`

A FastAPI-based AI agent that completes accounting tasks in Tripletex via a `/solve` endpoint. Uses Claude to parse prompts in 7 languages and executes the appropriate Tripletex API calls.

### Task 3 — NorgesGruppen Data (Object Detection)
`/ngdata`

Object detection submission for the NorgesGruppen Data challenge.
